Консольная утилита на Python для автоматизированной обработки и валидации структурированных данных из текстовых файлов. Программа выполняет интеллектуальный анализ формата входных данных, автоматически определяя разделители полей, 
типы данных столбцов и наличие заголовков. Осуществляет комплексную проверку корректности каждой записи: контролирует количество полей, соответствие типов данных, отсутствие пустых значений и допустимость числовых диапазонов. 
Все записи разделяются на корректные и содержащие ошибки с детализацией по каждой проблеме. Результатом работы является структурированный текстовый отчет, содержащий общую статистику (количество обработанных, валидных и невалидных записей) 
и подробный список всех обнаруженных ошибок с указанием номера строки, исходных данных и конкретной причины. Поддерживает кодировку UTF-8 и предоставляет интерактивный интерфейс с возможностью ручной коррекции автоматически определенных параметров перед началом обработки.
